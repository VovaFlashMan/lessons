Відпрацювання уроку №12(властивість: transform)

2D

translate (x, y) цей блок має властивість:transform: translate(-30px, 50px) при:hover; Такі самі відступи можна було б задати окремо кожній осі написавши так:transform: translateY(50px); transform: translateX(-30px); Але є одне але спрацює тільки останній запис по коду.

Scale (x, y) цей блок має властивість:transform:transform: scale(0.7); при:hover;Таке саме масштабування можна було б задати окремо кожній осі написавши так:transform: scale(0.7, 0.7);(Значення можуть бути при окремомцу записі різними наприклад:transform: scale(0.7, 0.9) )

Rotate (n + deg) цей блок має властивість: transform: rotate(45deg); при:hover;Таке саме провертання блоку Не можна! було б задати так: transform: rotateX(45deg); transform: rotateY(45deg) тому що виконуватиметься тільки останній запис;

Skew (x + deg, y + deg) цей блок має властивість:transform: skew(15deg, 25deg);при:hover;Таке саме провертання блоку Не можна! було б задати так: transform: skewX(15deg); transform:skewY(25deg); тому що виконуватиметься тільки останній запис;

Множинні 2D трансформації значення розділяються пробілом цей блок має властивість: transform: translate(-30px, 50px) scale(0.7) rotate(45deg);при:hover;Таке саме провертання блоку. Можна було б задати записувавши все по черзі як в перших трьох блоках(але навіщо робити зайві строчки коду коли можна записати все в одну).

Transform-origin цей блок має властивість: transform-origin: 100% 0%;при:hover додається властивість transform: rotate(25deg); Тобто маємо блок який при ховері прокручується вже не в стандартній позиції, а провертається в місці 100% 0% (можна задавти ті самі значення як для background /* transform-origin: top right; */);

3D

Властивості для 3D трансформації(значення просто для прикладу вписані):perspective-origin: center; perspective: 400px;Обов'язково мають бути задані батьківському елемнту!.

perspective-origin: center;(за замовчанням) perspective- не можна вказувати у відстотках

translate (x, y, z) цей блок має властивість:transform: transform: translate3D(-30px, 50px, 30px); при:hover; Такі самі відступи можна було б задати окремо кожній осі написавши так: transform: translateX(-30px);transform: translateY(50px); transform: translateZ(30px);Але є одне але спрацює тільки останній запис по коду.

Scale (x, y, z) цей блок має властивість:transform:transform: scale(0.7); при:hover;Таке саме масштабування можна було б задати окремо кожній осі написавши так:transform: scale(0.7, 0.7, 2);(Значення можуть бути при окремомцу записі різними наприклад:transform: scale(0.7, 0.9, 2); )

Rotate3d (1,1,1,0deg)- значення за замовчанням. Цей блок має властивість: transform: rotate3d(0.7, 0.5, 1, 45deg); при:hover

Множинні 3D трансформації цей блок має властивість: transform: translate3d(-30px, 50px, 30px) scale3d(0.7, 0.9, 2) rotate3d(0.7, 0.5, 1, 45deg);при:hover;Таке саме провертання блоку. Можна було б задати записувавши все по черзі як в перших трьох блоках(але навіщо робити зайві строчки коду коли можна записати все в одну).
